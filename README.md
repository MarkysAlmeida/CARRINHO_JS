# CARRINHO_JS
# 🛒 Carrinho de Compras em JavaScript

Projeto desenvolvido para praticar JavaScript puro (Vanilla JS), manipulando o DOM, criando objetos e aplicando regras de negócio em um carrinho de produtos.

---

## 📌 Sobre o Projeto

O sistema permite:

- Cadastrar produtos com:
  - Nome
  - Valor
  - Opção de desconto
- Adicionar produtos a uma lista dinâmica
- Calcular automaticamente:
  - Subtotal
  - Desconto condicional
  - Total final

---

## 🧠 Regra de Negócio

- Caso o subtotal ultrapasse **R$150,00**
- Apenas os produtos marcados como "elegíveis para desconto"
- Recebem **10% de desconto**
